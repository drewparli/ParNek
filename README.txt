Usage: java -jar parnek.jar <xesfile> [args...]

Arguments include:
   -all      run all the possible miners
   -f1       run ParNek's extension where excluded activity can be included by only one activity
   -fa       run ParNek's extension where excluded activity can be included by all the activities
   -cond     mine additionl conditions
   -opt      optimize exclusion relations (only combined with -f1 or -fa)
   -check    check all the activities while searching for possible exclusions (only combined with -f1 or -fa)
   -xml      save a graph to XML file

   -help     print this help message
   -h        print this help message
   -?        print this help message
